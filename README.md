# Audio Sample Searcher

A TypeScript project for analyzing audio files, creating embeddings using the CLAP model, and performing semantic search on audio samples.

## Features

- Audio file analysis using ONNX CLAP model
- Embedding generation and storage
- Semantic search capabilities for audio files

## Project Structure

```
.
├── assets/           # Contains the CLAP model and other assets
├── src/
│   ├── scripts/     # Analysis and search scripts
│   ├── utils/       # Utility functions
│   └── types/       # TypeScript type definitions
├── dist/            # Compiled JavaScript output
└── package.json     # Project dependencies and scripts
```

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the project:
   ```bash
   npm run build
   ```
